La couche middleware est la couche technique, liant les éléments d'infrastructure aux éléments fonctionnels. 

Cette couche regroupe notamment:

### Plateforme d'exécution
- Microfoft .NET
- Java Virtual Machine
- Microsfot IIS
- Apache Tomcat
- Python
- Node.js

### Plateforme de déploiement applicative
- Kubernetes
- Docker
- Machine virtuelle 

### Plateforme d'orchestration
- Apache Airflow
- Microsoft BizTalk

### Plateforme d'industrialisation
- Microsoft Azure DevOps
- GitLab
- Harbor
