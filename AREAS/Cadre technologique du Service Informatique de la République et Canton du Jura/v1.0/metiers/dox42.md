# Dox42
<!-- description/short {start} -->
dox42 est une solution d’automatisation permettant de générer des documents dynamiques et personnalisés à partir de multiples sources de données, intégrée dans les environnements Microsoft.
<!-- description/short {end} -->

## Description
Dox42 est une solution de **génération de documents automatisée** permettant de créer des documents dynamiques à partir de différentes sources de données (SharePoint, SAP, bases de données, etc.). Elle s’intègre dans les environnements Microsoft (Word, Excel, Power Automate) et facilite **la production de rapports, contrats ou formulaires personnalisés**. Dox42 réduit les erreurs manuelles et accélère les processus documentaires en entreprise.
