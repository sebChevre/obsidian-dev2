# Certus

<!-- description/short {start} -->
La plateforme Certus est une plateforme permettant **la signature des documents** basée sur la **blockchain KSI**. Elle intégère un workflow de validation des documents soumis et ce jusqu'à l'impression physique du document. Le code QR est ensuite vérifiable via une application mobile.

## Description
La plateforme Certus est une plateforme permettant **la signature des documents** basée sur la **blockchain KSI**. Elle intégère un workflow de validation des documents soumis et ce jusqu'à l'impression physique du document. Le code QR est ensuite vérifiable via une application mobile.
