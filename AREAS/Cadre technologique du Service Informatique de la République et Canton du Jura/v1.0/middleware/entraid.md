# Microsoft Entra ID 
<!-- description/short {start} -->
Microsoft Entra ID est une plateforme d’identité cloud assurant la gestion centralisée des utilisateurs, des accès et des applications SaaS.  
Elle étend l’annuaire Active Directory de la RCJU vers le cloud et fournit des services d’authentification modernes et sécurisés pour l’environnement hybride.  
<!-- description/short {end} -->

## Description  
Microsoft Entra ID (anciennement Azure Active Directory) est une **plateforme d’identité et d’accès cloud** qui étend les capacités d’Active Directory vers les environnements SaaS et hybrides.  

Elle offre des fonctions avancées d’**authentification moderne (OAuth2, OpenID Connect)**, de **Single Sign-On (SSO)**, de **gestion des appareils**, et de **contrôle d’accès conditionnel**.  
Entra ID est synchronisé avec l’annuaire interne via **Azure AD Connect**, garantissant une identité unique et cohérente pour les utilisateurs de la RCJU.  

La solution permet d’intégrer les services **Microsoft 365**, **Power Platform**, ainsi que d’autres applications tierces, tout en assurant une **gouvernance des identités et des accès** conforme aux standards de sécurité.  

Entra ID représente la **brique centrale d’identité cloud** de la RCJU, assurant la continuité de l’identité entre les systèmes on-premise et les services hébergés dans le cloud.
