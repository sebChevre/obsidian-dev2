# Plateforme Java Standard Edition

## Description
**Java SE (Java Platform, Standard Edition)** est la **plateforme de base du langage Java**, conçue pour le développement et l’exécution d’applications portables et multiplateformes.  
Elle fournit le **runtime Java (JRE)**, le **compilateur (javac)** et un ensemble complet de **bibliothèques standard** pour la gestion des entrées/sorties, de la mémoire, des threads, de la sécurité et des interfaces graphiques.  
Java SE constitue le socle des autres éditions comme **Java EE (Jakarta EE)** et **Java ME**, servant à la création d’applications d’entreprise ou embarquées.  
Sa philosophie "Write Once, Run Anywhere" garantit la **portabilité du code** sur différents systèmes d’exploitation et environnements matériels.  
Dans le contexte de la RCJU, Java SE est utilisé pour des **applications métiers robustes et pérennes**, nécessitant **fiabilité, sécurité et compatibilité interplateformes**.
